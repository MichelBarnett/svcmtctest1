# Marine Maintenance Technology: Troubleshooting Guides

Welcome to the troubleshooting guides repository for the Marine Maintenance Technology program!

This repo primarily contains the following content:

- [Troubleshooting Guides](troubleshooting/introduction.md) - A list of guides created to help you troubleshoot common issues you may face as a marine maintenance technician.

The repo also includes the following supporting content:

- [How to Contribute](contribute/introduction.md) - Details how to contribute new or modified content to this repo.
- [About](about/introduction.md) - Detail on how this repo was created.

Most of you will likely be interested in jumping right to the [Troubleshooting Guides](troubleshooting/introduction.md).  However, if you're interested in contributing changes or learning how the repo was created, check out the supporting links above.
